# canvas-demos

Interactive canvas demos built with Astro — particles, clock, and generative noise.

## Demos

| Page | Description |
|------|-------------|
| `/` | Hub — demo gallery |
| `/particle` | Canvas 2D particle simulation reacting to mouse/touch |
| `/clock` | SVG analog clock with timezone switching |
| `/noise` | Perlin noise generative art with real-time controls |

## Stack

- [Astro](https://astro.build/) — static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
- [Motion](https://motion.dev/) — animations
- GitHub Actions — CI/CD
- GitHub Pages — hosting

## Commands

| Command | Action |
|---------|--------|
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server at `localhost:4321` |
| `npm run build` | Build to `./dist/` |
| `npm run preview` | Preview build locally |
